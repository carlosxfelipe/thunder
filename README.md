# Thunar

Gerenciador de arquivos para macOS escrito em Swift com SwiftUI.

## Requisitos

- macOS 14.0 (Sonoma) ou superior
- Xcode 15 ou superior

## Como rodar

```
git clone https://github.com/carlosxfelipe/thunar.git
cd thunar
open thunar.xcodeproj
```

No Xcode, selecione o target `thunar` e clique em Run (Cmd+R).

## Atalhos de teclado

| Atalho | Acao |
|---|---|
| Cmd+C | Copiar |
| Cmd+X | Recortar |
| Cmd+V | Colar |
| Cmd+T | Nova aba |
| Cmd+W | Fechar aba |
| Ctrl+Tab | Proxima aba |
| Ctrl+Shift+Tab | Aba anterior |
| Space | Quick Look |
| Enter | Abrir item (modo icones) |
| Setas | Navegar entre itens (modo icones) |
| Shift+Setas | Selecao multipla (modo icones) |
| Shift+Clique | Selecao em bloco (modo icones) |
| Cmd+Clique | Selecao individual |
| Letras/Numeros | Saltar para item pelo nome |
